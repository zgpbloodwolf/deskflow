<objective>
Remove the SecurityLevel enum from all interfaces and eliminate all dispatch paths across the codebase.

Purpose: The SecurityLevel enum (PlainText/Encrypted/PeerAuth) is the central dispatch mechanism for TLS. Removing it from ISocketFactory and all callers makes the entire TLS layer unreachable. The AsioTCPSocketFactory already ignores SecurityLevel (marked [[maybe_unused]]), so this is purely interface cleanup.

Output: All factory interfaces, client, server, and listener code operate without any SecurityLevel parameter. The TCPSocketFactory is cleaned to only create plain-text sockets (it must compile since it is still in the build).
</objective>
