<objective>
Implement lock-free SPSC event buffers that separate keyboard FIFO queue from mouse atomic latest-position slot, and integrate the buffers into AsioTCPSocket.

Purpose: Per D-05, the input processing thread (EventQueue main thread, producer) and network I/O thread (Asio io_context, consumer) are decoupled through lock-free SPSC buffers. Mouse move events overwrite old values on write (D-08 coalescing), keyboard events go through FIFO preserving order and completeness. Per D-10, key state table tracks all pressed keys.

Output: SpscQueue.h (SPSC FIFO + MousePositionSlot), InputEventBuffer (composite buffer), KeyStateTable (key state table). AsioTCPSocket integrates InputEventBuffer and KeyStateTable, with 200Hz steady_timer reading mouse position and immediate keyboard FIFO draining.

IMPORTANT: This plan only creates the SPSC buffer infrastructure and wires it inside AsioTCPSocket. The upper-layer ClientProxy1_0::mouseMove()/keyDown()/keyUp() call path modification to actually USE the SPSC buffer is in Plan 03 Task 2.

**Step A: Create src/lib/net/SpscQueue.h**

New file containing two components:

1. **MousePositionSlot** -- atomic store/load of latest mouse position:
```cpp
struct alignas(64) MousePositionSlot
{
  std::atomic<int32_t> x{0};
  std::atomic<int32_t> y{0};
  std::atomic<bool> updated{false};

  void store(int32_t newX, int32_t newY) noexcept {
    x.store(newX, std::memory_order_relaxed);
    y.store(newY, std::memory_order_relaxed);
    updated.store(true, std::memory_order_release);
  }

  bool load(int32_t &outX, int32_t &outY) noexcept {
    if (!updated.exchange(false, std::memory_order_acquire)) {
      return false;
    }
    outX = x.load(std::memory_order_relaxed);
    outY = y.load(std::memory_order_relaxed);
    return true;
  }
};
```
Memory ordering (per RESEARCH.md Pitfall 5): Writer stores x/y relaxed first, then stores updated flag with release. Reader exchanges updated flag with acquire, then loads x/y relaxed.

2. **SpscFifo<T, Capacity>** -- single-producer single-consumer ring buffer:
```cpp
template <typename T, size_t Capacity>
class alignas(64) SpscFifo
{
  static constexpr size_t kMask = Capacity - 1;
  static_assert((Capacity & kMask) == 0, "Capacity must be power of 2");

  std::array<T, Capacity> buffer_{};
  alignas(64) std::atomic<size_t> head_{0};
  alignas(64) std::atomic<size_t> tail_{0};

public:
  bool push(const T &item) noexcept { /* RESEARCH.md Pattern 3 */ }
  bool pop(T &item) noexcept { /* RESEARCH.md Pattern 3 */ }
  bool empty() const noexcept { /* head == tail */ }
};
```
- Capacity default 256 (meets RESEARCH.md Pitfall 6 recommendation for keyboard events)
- `push()` uses memory_order_relaxed for tail read, memory_order_acquire for head read, memory_order_release for tail write
- `pop()` uses memory_order_relaxed for head read, memory_order_acquire for tail read, memory_order_release for head write
- head and tail each alignas(64) to prevent false sharing (RESEARCH.md Pitfall 2)

File header: standard Deskflow SPDX header. Use `#pragma once`.

```cpp
struct KeyboardEvent {

class InputEventBuffer

public:

  // Consumer pops keyboard event (from Asio io_context thread)

  // Is keyboard queue empty
  bool isKeyboardEmpty() const noexcept;

private:
```

```cpp
#pragma once
#include "deskflow/KeyTypes.h"
#include <cstdint>
#include <mutex>
#include <vector>
#include <unordered_map>

struct KeyReleaseInfo {
  KeyID key;
  KeyModifierMask mask;
  KeyButton button;
};

class KeyStateTable
{
public:
  // Record keyDown (called from EventQueue thread)
  void recordKeyDown(KeyID key, KeyModifierMask mask, KeyButton button);

  // Record keyUp (called from EventQueue thread)
  void recordKeyUp(KeyID key);

  // On disconnect: iterate all pressed keys, return release list, then clear
  // Called from Asio io_context thread (in disconnect callback)
  std::vector<KeyReleaseInfo> releaseAll();

  // Are any keys currently pressed
  bool hasPressedKeys() const;

private:
  struct KeyInfo { KeyModifierMask mask; KeyButton button; };
  mutable std::mutex m_mutex; // protects: EventQueue thread writes, Asio thread reads (on disconnect)
  std::unordered_map<KeyID, KeyInfo> m_pressedKeys;
};
```

Implementation notes:
- `recordKeyDown()`: lock, m_pressedKeys[key] = {mask, button}
- `recordKeyUp()`: lock, m_pressedKeys.erase(key)
- `releaseAll()`: lock, iterate m_pressedKeys building return list, then clear()
- Uses mutex instead of lock-free: only read from Asio thread on disconnect (low frequency), normal path recordKeyDown/Up only writes from EventQueue thread, mutex overhead negligible

2. **startMouseSendTimer() implementation (200Hz):**
```cpp
void AsioTCPSocket::startMouseSendTimer()
{
  m_mouseTimer.expires_after(std::chrono::milliseconds(5)); // 200Hz = 5ms interval
  m_mouseTimer.async_wait(
    asio::bind_executor(m_strand,
      [this, self = shared_from_this()](const std::error_code &ec) {
        onMouseTimer(ec);
      }
    )
  );
}

void AsioTCPSocket::onMouseTimer(const std::error_code &ec)
{
  if (ec) return; // timer cancelled (normal shutdown)

  int32_t x, y;
  if (m_eventBuffer.loadMousePosition(x, y)) {
    // New mouse position available, send DMMV via ProtocolUtil
    ProtocolUtil::writef(this, kMsgDMouseMove, x, y);
  }

  // Reschedule timer
  startMouseSendTimer();
}
```

3. **drainKeyboardFifo() implementation:**
```cpp
void AsioTCPSocket::drainKeyboardFifo()
{
  // Runs on io_context thread (protected by strand)
  KeyboardEvent evt;
  while (m_eventBuffer.popKeyboardEvent(evt)) {
    sendKeyboardEvent(evt);
  }

  // Schedule next check via short timeout timer (1ms) instead of busy-wait
  m_keyboardPollTimer.expires_after(std::chrono::milliseconds(1));
  m_keyboardPollTimer.async_wait(
    asio::bind_executor(m_strand,
      [this, self = shared_from_this()](const std::error_code &ec) {
        if (!ec) drainKeyboardFifo();
      }
    )
  );
}
```

4. **sendKeyboardEvent() implementation:**
```cpp
void AsioTCPSocket::sendKeyboardEvent(const KeyboardEvent &evt)
{
  switch (evt.type) {
  case KeyboardEvent::Type::Down:
    m_keyState.recordKeyDown(evt.key, evt.mask, evt.button);
    ProtocolUtil::writef(this, kMsgDKeyDownLang, evt.key, evt.mask, evt.button, &evt.language);
    break;
  case KeyboardEvent::Type::Up:
    m_keyState.recordKeyUp(evt.key);
    ProtocolUtil::writef(this, kMsgDKeyUp1_0, evt.key, evt.mask);
    break;
  case KeyboardEvent::Type::Repeat:
    ProtocolUtil::writef(this, kMsgDKeyRepeat1_0, evt.key, evt.mask, 1);
    break;
  }
}
```

| EventQueue thread -> SPSC buffer | Producer writes mouse position and keyboard events using lock-free atomic operations |
