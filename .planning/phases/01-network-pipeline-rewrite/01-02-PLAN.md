**Step A: 创建 src/lib/net/SpscQueue.h**

新文件，包含两个组件：

1. **MousePositionSlot** -- 原子操作存储/读取最新鼠标位置：
```cpp
struct alignas(64) MousePositionSlot
{
  std::atomic<int32_t> x{0};
  std::atomic<int32_t> y{0};
  std::atomic<bool> updated{false};

  void store(int32_t newX, int32_t newY) noexcept {
    x.store(newX, std::memory_order_relaxed);
    y.store(newY, std::memory_order_relaxed);
    updated.store(true, std::memory_order_release);
  }

  bool load(int32_t &outX, int32_t &outY) noexcept {
    if (!updated.exchange(false, std::memory_order_acquire)) {
      return false;
    }
    outX = x.load(std::memory_order_relaxed);
    outY = y.load(std::memory_order_relaxed);
    return true;
  }
};
```
Memory ordering 说明（按 RESEARCH.md Pitfall 5）：写入端先 relaxed store x/y，再 release store updated flag。读取端先 acquire exchange updated flag，再 relaxed load x/y。

2. **SpscFifo<T, Capacity>** -- 单生产者单消费者环形缓冲区：
```cpp
template <typename T, size_t Capacity>
class alignas(64) SpscFifo
{
  static constexpr size_t kMask = Capacity - 1;
  static_assert((Capacity & kMask) == 0, "Capacity must be power of 2");

  std::array<T, Capacity> buffer_{};
  alignas(64) std::atomic<size_t> head_{0};
  alignas(64) std::atomic<size_t> tail_{0};

public:
  bool push(const T &item) noexcept { /* RESEARCH.md Pattern 3 */ }
  bool pop(T &item) noexcept { /* RESEARCH.md Pattern 3 */ }
  bool empty() const noexcept { /* head == tail */ }
};
```
- Capacity 默认 256（满足 RESEARCH.md Pitfall 6 对键盘事件的容量建议）
- `push()` 使用 memory_order_relaxed 读 tail，memory_order_acquire 读 head，memory_order_release 写 tail
- `pop()` 使用 memory_order_relaxed 读 head，memory_order_acquire 读 tail，memory_order_release 写 head
- head 和 tail 各自 alignas(64) 防止 false sharing（RESEARCH.md Pitfall 2）

文件头：标准 Deskflow SPDX header。使用 `#pragma once`。

**Step C: 创建 src/lib/net/KeyStateTable.h / .cpp**

按键状态追踪表（RESEARCH.md Pattern 5）：

```cpp
#pragma once
#include "deskflow/KeyTypes.h"
#include <cstdint>
#include <mutex>
#include <vector>
#include <unordered_map>

struct KeyReleaseInfo {
  KeyID key;
  KeyModifierMask mask;
  KeyButton button;
};

class KeyStateTable
{
public:
  // 记录 keyDown（EventQueue 线程调用）
  void recordKeyDown(KeyID key, KeyModifierMask mask, KeyButton button);

  // 记录 keyUp（EventQueue 线程调用）
  void recordKeyUp(KeyID key);

  // 断连时调用：遍历所有 pressed keys，返回释放列表，然后清空
  // Asio io_context 线程调用（断连回调中）
  std::vector<KeyReleaseInfo> releaseAll();

  // 是否有按键处于按下状态
  bool hasPressedKeys() const;

private:
  struct KeyInfo { KeyModifierMask mask; KeyButton button; };
  mutable std::mutex m_mutex; // 保护：EventQueue 线程写，Asio 线程读（断连时）
  std::unordered_map<KeyID, KeyInfo> m_pressedKeys;
};
```

实现说明：
- `recordKeyDown()`: 加锁，m_pressedKeys[key] = {mask, button}
- `recordKeyUp()`: 加锁，m_pressedKeys.erase(key)
- `releaseAll()`: 加锁，遍历 m_pressedKeys 构建返回列表，然后 clear()
- 使用 mutex 而非 lock-free：因为只有断连时才在 Asio 线程读（低频操作），正常路径中 recordKeyDown/Up 只在 EventQueue 线程写，mutex 开销可忽略

2. **startMouseSendTimer() 实现（200Hz）：**
```cpp
void AsioTCPSocket::startMouseSendTimer()
{
  m_mouseTimer.expires_after(std::chrono::milliseconds(5)); // 200Hz = 5ms interval
  m_mouseTimer.async_wait(
    asio::bind_executor(m_strand,
      [this, self = shared_from_this()](const std::error_code &ec) {
        onMouseTimer(ec);
      }
    )
  );
}

void AsioTCPSocket::onMouseTimer(const std::error_code &ec)
{
  if (ec) return; // timer 被取消（正常关闭）

  int32_t x, y;
  if (m_eventBuffer.loadMousePosition(x, y)) {
    // 有新的鼠标位置，通过 ProtocolUtil 发送 DMMV
    ProtocolUtil::writef(this, kMsgDMouseMove, x, y);
  }

  // 重新调度定时器
  startMouseSendTimer();
}
```

3. **drainKeyboardFifo() 实现：**
```cpp
void AsioTCPSocket::drainKeyboardFifo()
{
  // 在 io_context 线程中运行（通过 strand 保护）
  KeyboardEvent evt;
  while (m_eventBuffer.popKeyboardEvent(evt)) {
    sendKeyboardEvent(evt);
  }

  // 使用 post 调度下一次检查（非定时器驱动，写完即检查）
  // 但为了避免忙等，使用短超时定时器（1ms）
  m_keyboardPollTimer.expires_after(std::chrono::milliseconds(1));
  m_keyboardPollTimer.async_wait(
    asio::bind_executor(m_strand,
      [this, self = shared_from_this()](const std::error_code &ec) {
        if (!ec) drainKeyboardFifo();
      }
    )
  );
}
```

4. **sendKeyboardEvent() 实现：**
```cpp
void AsioTCPSocket::sendKeyboardEvent(const KeyboardEvent &evt)
{
  switch (evt.type) {
  case KeyboardEvent::Type::Down:
    m_keyState.recordKeyDown(evt.key, evt.mask, evt.button);
    ProtocolUtil::writef(this, kMsgDKeyDownLang, evt.key, evt.mask, evt.button, &evt.language);
    break;
  case KeyboardEvent::Type::Up:
    m_keyState.recordKeyUp(evt.key);
    ProtocolUtil::writef(this, kMsgDKeyUp1_0, evt.key, evt.mask);
    break;
  case KeyboardEvent::Type::Repeat:
    ProtocolUtil::writef(this, kMsgDKeyRepeat1_0, evt.key, evt.mask, 1);
    break;
  }
}
```
